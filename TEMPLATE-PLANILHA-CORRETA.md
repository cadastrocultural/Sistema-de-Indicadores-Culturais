## 📝 COMO USAR ESTE TEMPLATE:

### **OPÇÃO 1: Substituir dados da planilha existente**
1. Abra sua planilha com os 98 projetos
2. **Delete as 74 linhas** dos projetos NÃO aprovados
3. **Adicione a coluna "Valor"** com os valores acima
4. **Adicione a coluna "status"** com "aprovado" em todas

### **OPÇÃO 2: Criar planilha nova do zero**
1. Crie uma nova planilha Excel
2. Copie o cabeçalho acima
3. Cole os 24 projetos do exemplo
4. **Substitua os nomes** pelos nomes reais do PDF
5. Salve como "PEC_2024_Aprovados.xlsx"
